# CinemaApp — WPF приложение для бронирования билетов

Полноценное WPF-приложение для онлайн-бронирования билетов в кинотеатр.

## Стек технологий

- **.NET 8** + **WPF**
- **Entity Framework Core 8** + SQL Server / LocalDB
- **CommunityToolkit.Mvvm** (MVVM паттерн)
- Архитектура: **MVVM** (Model–View–ViewModel)

## Функциональность

| Раздел | Описание |
|--------|----------|
| **Афиша** | Каталог фильмов с постерами, поиск, фильтры по жанру, рейтинг |
| **Детали фильма** | Описание, каст, режиссёр, расписание сеансов на сегодня |
| **Выбор мест** | Интерактивная схема зала, выбор мест, расчёт стоимости |
| **Мои билеты** | Список предстоящих билетов, история, QR-код |
| **Профиль** | Вход, регистрация, программа лояльности |

## Быстрый старт

### Требования

- Visual Studio 2022 (или новее)
- .NET 8 SDK
- SQL Server LocalDB (устанавливается с Visual Studio)

### Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd CinemaApp
   ```

2. Откройте `CinemaApp.sln` в Visual Studio

3. Восстановите пакеты: правой кнопкой на Solution → **Restore NuGet Packages**

4. Запустите проект (F5)

   При первом запуске база данных создаётся и заполняется тестовыми данными автоматически.

### Строка подключения

По умолчанию используется **LocalDB**:

```
Server=(localdb)\MSSQLLocalDB;Database=CinemaDb;Trusted_Connection=True;
```

Если хотите использовать полноценный SQL Server — отредактируйте строку в `Data/CinemaDbContext.cs`.

## Структура проекта

```
CinemaApp/
├── Models/           # Модели данных (Movie, Session, Seat, Ticket, User)
├── Data/             # DbContext и инициализатор БД с тестовыми данными
├── ViewModels/       # MVVM ViewModel для каждого экрана
├── Views/            # XAML-интерфейс (UserControl для каждого раздела)
└── Converters/       # Value converters для привязок данных
```
